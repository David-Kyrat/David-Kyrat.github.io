Exercice 7, Matrices diagonales

Pour cet exercice, nous allons utiliser de différentes manière la fonction diag.

n = 5; A = rand(n)

A =

    0.8687    0.4314    0.1361    0.8530    0.0760
    0.0844    0.9106    0.8693    0.6221    0.2399
    0.3998    0.1818    0.5797    0.3510    0.1233
    0.2599    0.2638    0.5499    0.5132    0.1839
    0.8001    0.1455    0.1450    0.4018    0.2400


B = diag(diag(A))

B =

    0.8687         0         0         0         0
         0    0.9106         0         0         0
         0         0    0.5797         0         0
         0         0         0    0.5132         0
         0         0         0         0    0.2400

Appeler une fois diag retourne les éléments sur la diagonale principale de A sous la forme d'un vecteur et appeler diag sur ce dernier résultat construit une matrice diagonale dont la diagonale est composé de précisément ces éléments-ci.
